# GCP Simulator HMI
 HMI application for controlling GCP simulator hardware.
 Runs as a locally hosted web app, accessed through the browser.

## Requirements
- `pip install nicegui`
- `pip install numpy`
- `pip install pywebview`
- `pip install pyserial`
